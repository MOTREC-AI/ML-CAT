There are four sets of code in this folder.

Cross_Validation_AllData - Processes all our data for cross validation (includes KS patients)
Cross_Validation_KSData - Processes just KS patient data for cross validation
Test_DataSet - Processes just the Test data (includes KS patients)
Test_DataSet_KS - Processes the Test data using only KS patient data

Launch the code with the respective run_model.sh scripts.
